### Configurando o Phishing no Kali Linux
- Execute o  Dark-Phish
```bash
python3 dark-phish.py
```
- Para ajuda e informações de uso:
```bash
python3 dark-phish.py -h
```
- Para acessar as credenciais salvas:
```bash
python3 dark-phish.py -r
```
